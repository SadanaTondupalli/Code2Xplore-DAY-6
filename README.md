# Code2Xplore Day-6 Challenge

## Challenge Title
Smart Playlist Intelligence System

## Problem Understanding
This program analyzes a playlist using song durations. It validates input, categorizes the playlist, and provides recommendations.

## Logic Used
The program checks for invalid durations, calculates total duration and number of songs, detects repetition, and classifies the playlist based on defined rules.

## Sample Output
Total Duration: 360 seconds  
Songs: 3  
Category: Repetitive Playlist  
Recommendation: Add variety

## Learning Outcome
This task improved my understanding of list handling, validation, and logical decision-making in Python.
